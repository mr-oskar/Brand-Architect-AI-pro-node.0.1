## Fix Prompt

**Subject: Immediate Action Required: Critical Security Vulnerability Fixes**

As a Senior Security-Focused Full Stack Developer, you are tasked with implementing immediate and comprehensive security fixes for this project based on a recent security audit. Please execute the following tasks with high precision:

### 1. API Protection & Authorization (Highest Priority)
*   Review all endpoints within the `/api/` directory.
*   Ensure all administrative paths (e.g., `/api/admin/*`) and sensitive data paths (e.g., `/api/dashboard/summary`, `/api/brands`) are strictly protected by Authentication and Role-Based Access Control (RBAC) middleware.
*   Unauthenticated users or users without the `admin` role must be strictly prohibited from accessing user data, system logs, or usage statistics.

### 2. Server Hardening (Security Headers & CORS)
*   Install and configure the `helmet` library (if Node.js) to enable essential security headers.
*   Implement a strict `Content-Security-Policy` (CSP) and enable `X-Frame-Options: DENY` to prevent Clickjacking attacks.
*   Configure `CORS` to allow only trusted origins to access the API.

### 3. Endpoint Protection (Rate Limiting)
*   Implement `Rate Limiting` on all authentication endpoints, specifically `/api/auth/login` and `/api/auth/register`, to mitigate Brute-force and Credential Stuffing attacks.

### 4. Data & File Leakage Prevention
*   Ensure the server does not serve sensitive files such as `.env`, `.git`, `package.json`, or source code files (`.ts`, `.tsx`) to the public.
*   Configure the server to return a `404 Not Found` or `403 Forbidden` status for these files instead of redirecting to the home page.

### 5. Frontend Sanitization
*   Scan the React/Frontend codebase for any hardcoded API keys or secrets.
*   Move all sensitive credentials to environment variables (`.env`) and ensure they are only handled through the backend.

**Requirement:** Implement these fixes while ensuring the system remains fully functional. Provide a summary of the security enhancements once completed.
